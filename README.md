# more-cup

A mobile-friendly web game with tap-and-hold liquid pouring physics.

## How to Play

1. Open `index.html` in a browser.
2. Press and hold anywhere on the screen to pour water from that touch point.
3. Move your finger while holding to change the pour location.
4. Watch the liquid interact with the platform placed one-third above the bottom of the screen.

## Files

- `index.html` - game page and HUD.
- `style.css` - responsive styling and visual theme.
- `game.js` - Matter.js world setup, liquid particle emission, and touch controls.
